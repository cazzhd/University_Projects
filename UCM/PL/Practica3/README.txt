Compilar JAVACC:
1.a Se crea un nuevo proyecto en eclipse.
2. Instalamos el pluggin de JavaCC (y reiniciamos)
3. Nos vamos a propiedades del proyecto > JavaCC Options y elegimos el archivo JavaCC jar.

1.b De manera similar, podemos realizar el siguiente comando:
java -classpath <ruta>/javacc.jar javacc <archivo>.jj
